# Cosmonus — AI Solutions (Next.js)

Minimal Next.js (App Router) starter for Cosmonus — a company offering AI solutions.

Quick start

```bash
npm install
npm run dev
```

Build for production

```bash
npm run build
npm start
```

Deploy to Vercel: connect this repository to Vercel and push — Vercel will detect Next.js automatically.

Notes
- App Router (Next.js 14+) is used under `app/`.
- No backend logic yet — frontend-only scaffold.
- Bootstrap is included via npm.
